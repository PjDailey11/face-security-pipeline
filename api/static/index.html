Face Security Pipeline

This is an on-device face security pipeline. It detects faces in an image/video frame, optionally checks liveness, and can verify identity by comparing to people you enroll locally (no cloud biometric storage).

Quick start: (1) Load an image (Paste / Pick / Webcam → Capture) → (2) Click Infer. To recognize someone by name: (3) type a name → (4) click Enroll on a clear face photo → (5) Infer again.
Important: If you haven't trained a face detector yet, the server may be using yolov8n.pt (COCO), which detects person / tie rather than face. For real face boxes you need a fine-tuned weights/yolov8_face.pt.
API: checking… Open API docs

Paste: click anywhere in this box and press Ctrl+V. Drag & drop: drop an image file here. Webcam: click Use webcam, then click Capture (or click the image area).

Enroll stores a local embedding in ChromaDB. Infer returns detections + identity match + similarity score. Liveness is shown only if an anti-spoof model is configured.
